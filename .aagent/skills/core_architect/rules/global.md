Global Agency Rules

1. Строгий Цикл Разработки (Context Engineering)

ЛЮБАЯ задача выполняется строго по циклу. Переход на следующий этап только после подтверждения пользователя:

Research (Анализ)

Design (Проектирование)

Plan (Пошаговый план)

Implement (Код)

2. Санитария Кода

При миграции с конструкторов (Тильда и др.) агент ОБЯЗАН вычистить все классы конструктора (t-rec, инлайн-стили).

Использовать только чистый HTML5 и Tailwind CSS.

Никаких заглушек вида "допишите логику сами". Код должен быть завершенным.

3. SEO Quality Gates

Правило 1500 символов: весь коммерческий текст должен быть в верхней части DOM-дерева.

Запрещена галлюцинация текстов.

Обязательно применение тегов H1-H3 и микроразметки Schema.org (JSON-LD) перед сдачей работы.
