Workflow: Premium Web Production Pipeline

Описание

Этот сценарий объединяет SEO-Архитектора, Core-Архитектора и Motion-Дизайнера в единую автоматизированную производственную линию. Применяется для сборки сайтов под ключ.

Правила маршрутизации (Routing)

При запуске этого пайплайна, система Antigravity обязана строго следовать очередности этапов. Переход к следующей фазе (Phase) происходит ТОЛЬКО после явного утверждения пользователем текущего результата.

Phase 1: Research & Blueprint (Агент: SEO & GEO Architect)

Входные данные: ТЗ от пользователя (тематика, тексты, референсы).

Задача: Сгенерировать семантический каркас. Внедрить правило 1500 символов, прописать иерархию H1-H3, создать код микроразметки JSON-LD.

Выход: Текстовый черновик структуры с комментариями для разработчика.

[Ожидание аппрува от CEO]

Phase 2: Implement Base (Агент: Core Architect)

Входные данные: Утвержденный SEO-каркас.

Задача: Написать чистый, валидный HTML-файл с использованием Tailwind CSS. Обеспечить Mobile-First адаптивность. Подготовить классы-заглушки для будущих анимаций.

Выход: Рабочий файл index.html с идеальной SEO-базой, но без сложного визуала.

[Ожидание аппрува от CEO]

Phase 3: Cinematic Polish (Агент: Motion & Interaction Designer)

Входные данные: Готовый index.html и пожелания по стилю.

Задача: Внедрить CSS-эффекты матового стекла (Liquid Glass), мягкие тени и GSAP-анимации (Parallax, Scrollytelling на Canvas, если есть секвенции).

Выход: Финальный, кинематографичный лендинг, готовый к публикации.
